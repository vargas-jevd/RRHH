1. Crear una base de datos sencilla llamada RRHH.

2. Crear una tabla llamada EMPLEADO con campos para ID, NOMBRE, APELLIDO, DI, FECHA_DE_NACIMIENTO, DEPARTAMENTO, FECHA_CONTRATACION y SALARIO.

3. Crear una aplicación en JAVA que permita:

	1. Crear empleados en la base de datos.

	2. Consultar uno o todos los empleados de la base de datos.
	
	3. Actualizar la información de un empleado en la base de datos.
	
	4. Eliminar un empleado de la base de datos.
	
La aplicación sera una aplicación por consola y basada en menús numerados, en donde el usuario introduce el número de la opción asociada a la acción que desea realizar y el programa le solicita los datos necesarios para ejecutar la acción o le muestra un sub-menú de ser necesario.

CRUD = Create, Retrieve, Update, Delete
